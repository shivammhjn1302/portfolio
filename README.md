# Shivam Mahajan — Cyberpunk Portfolio

A Next.js cyberpunk-style portfolio built from Shivam's resume.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
npm start
```

## Customize

Edit `app/page.tsx` for content and `app/globals.css` for styling/effects.
